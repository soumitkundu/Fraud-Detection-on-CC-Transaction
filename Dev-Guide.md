# Developer Guide - Fraud Transaction Detection on Credit Card

### Helpful commands to run the application

**Create Virtual Environment:**

```
py -m venv .venv
```

**Activate the Virtual Environment:**

_Windows_

```
.venv/Scripts/activate
```

_Linux/MacOS_

```
source venv/bin/activate
```

**To deactivate the Virtual Environment**

```
deactivate
```

**Upgrade pip version:**

```
python -m pip install --upgrade pip
```

**Install Required Libraries/Packages**

> Install the below libraries one-by-one with `pip install` command

```
requests, dotenv, fastapi, uvicorn, pandas, numpy, kagglehub, matplotlib, seaborn, scikit-learn, scipy, category_encoders, xgboost, jupyter or jupyterlab
```

**To Check The Installed Libraries**

```
pip list
```

**Freeze the `requirements.txt` file**

```
pip freeze > requirements.txt
```
